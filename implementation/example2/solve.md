# 시각

* N이 입력되면 00시 00분 00초부터 N시 59분 59초까지의 모든 시각 중에서 3이 하나라도 포함되는 모든 경우의 수
* 첫째줄에 정수 N(0 <= N <= 23) 입력

## 풀이

* 24시간은 86400초이므로 모든 시각을 체크하며 경수의 수를 구해도 문제되지 않는다.
* 초의 경우의 수 => 0부터 60까지 3이 포함되어 있는 경우의 수를 체크한다.
* 분의 경우의 수 => 초를 제외하고 분만 고려하면 `초의 경우의 수`와 동일하다.
* 분과 초를 합한 경우의 수 => `(초의 경우의 수) ^ 2`
* N이 0이어도 분과 초를 합한 경우의 수가 1번 발생한다.
* `N * ((초의 경우의 수) ^ 2)`를 구하면 된다.

### 논리 오류 발견

* `(초의 경우의 수) ^ 2`는 분에 3이 포함될 때의 경우의 수가 누락된다.
* `N * ((초의 경우의 수) ^ 2)`은 N에 3일 때의 경우의 수가 누락된다.
* `N + 1`이 아닌 `N`은 경우의 수가 누락된다.

### 코드 논리 재설계

* 0부터 60까지 3이 포함되어 있는 경우의 수를 체크하여 `1분 경우의 수`를 구한다.
* 분의 경우의 수를 3이 포함될 때와 안될 때 2가지로 분류하여 계산하여 `1시간 경우의 수`를 구한다.
  * 3이 포함되면 1분의 모든 경우의 수인 `60`을 더한다.
  * 3이 포함되지 않으면 이전에 구한 `1분 경우의 수`를 더한다.
* N의 경우의 수를 3이 포함될 때와 안될 때 2가지로 분류하여 계산한다.
  * 3이 포함되면 1시간의 모든 경우의 수인 `3600`을 더한다.
  * 3이 포함되지 않으면 이전에 구한 `1시간 경우의 수`를 더한다.
