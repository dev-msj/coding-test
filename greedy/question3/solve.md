# N이 1이 될 때까지

* N이 1이 될 때까지 2가지 과정 중 1개를 반복적으로 수행
* 2번째 연산은 N이 K로 나눠 떨어질 때만 가능
  1. N에서 1을 뺀다.
  2. N을 K로 나눈다.
* N과 K가 주어질 때 1, 2 과정을 수행할 최소 횟수
* 시간 제한: 1초, 메모리 제한 128MB
* 2 <= (N, K) <= 100,000
* N >= K

## 풀이

조건상 최대 케이스인 `N - 1`을 반복적으로 수행해도 시간 복잡도는 **O(N)**이다. 따라서 루프를 통해 K로 나눠 떨어지는지 매번 검사해도 문제 없다.
