# 위에서 아래로

* 시간 제한: 1초, 메모리 제한: 128MB
* 크기와 상관없이 나열된 수열 정렬
* 첫째줄: 수열의 크기 N(1<=N<=500)
* 둘째줄부터: N(1<=수<=100,000)개의 정수를 입력받음

## 풀이

* 정렬되지 않은 수열을 정렬하는 것이므로 퀵 정렬 알고리즘 사용, O(NlogN)

```text
if 수열의 길이가 1이하면:
  return

pivot = 수열의 0번째 값 추출
tail = 0번째를 제외한 나머지 수열

left_arr = 수열에서 pivot보다 작은 list 생성
right_arr = 수열에서 pivot보다 큰 list 생성

return 퀵정렬(right_arr) + [pivot] + 퀵정렬(left_arr)
```

### 놓친 점

크기와 상관없이 나열된 수열을 정렬하라는 글만 보고 퀵 정렬임을 확신했다.
하지만 이 확신 때문에 내림차순으로 정렬하라는 내용을 까먹었던 것 같다.
문제가 요구하는 내용을 정확히 파악하기 위해 꼼꼼히 읽고 메모하도록 하자.
