# 왕실의 나이트

* 시간 제한: 1초, 메모리 제한: 128MB
* 8*8 좌표 평면 위의 특정 한 칸에 나이트가 있음
* 나이트는 2가지의 이옹이 가능
  1. 수평 2칸, 수직 1칸
  2. 수직 2칸, 수평 1칸
* 나이트가 이동할 수 있는 모든 경우의 수를 출력
* 행은 `1 ~ 8`, 열은 `a ~ h`로 표현
* 입력 문자는 `a1`과 같이 열과 행으로 입력

## 풀이

* 고려해야 할 경우의 수는 `64 * 8 = 512`가지이므로 "O(N)"으로 풀이 가능
* `a ~ h` 문자를 `1 ~ 8`의 숫자로 변환하는 dict 구현
  * 열: `int(dict[input[0]])`
  * 행: `input[1]`
* 이동 가능한 경우의 수는 8가지
  * [[2, 1], [2, -1], [-2, 1], [-2, -1], [1, 2], [1, -2], [-1, 2], [-1, -2]]
* 행, 열 값에 이동 가능한 경우의 수를 더한 좌표값이 `1 ~ 8`의 범위를 벗어나는지 확인
* 벗어나지 않으면 카운트

## 오답 노트

오답이라기엔 애매하지만 C에서는 단일 문자는 char로 처리되는데, 파이썬에서는 모두 str로 취급되다보니 그냥 dict를 통해 숫자로 매칭시켰다.
하지만 답안지를 확인해본 결과 파이썬에서 `ord` 메서드를 활용해 단일 문자를 char로 활용할 수 있는 기능이 있었다.
그럼 `ord(intput[0]) - ord['a'] + 1`과 같이 코드를 작성하여 아스키 값을 활용해 숫자로 간단하게 변환할 수 있다.
